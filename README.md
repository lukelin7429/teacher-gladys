# Teacher Glad — Bilingual Teaching Portfolio

A rebuilt, animated portfolio for **Gladys Mabute (Teacher Glad)**, a foreign English
teacher at **Nanhsing Elementary School**, Taiwan, through the Taiwan Foreign English
Teacher Program (TFETP).

Mirrors the structure of her original Google Site:

- **Home**
- **Bilingual Initiative** — English Club · English Corner · International Sister School · Communication Arts
- **About Me**
- **Learning Activities**
- **Learning Outputs & Teaching Materials**

## Tech
Static HTML/CSS/JS. Pages are generated by `build.py`, which pulls photos dynamically
from `assets/img/` (named `<page-key>_NNN.jpg`). Interactive embeds (Wordwall, Padlet,
Canva), YouTube, and Google Drive videos are wired in directly.

Rebuild:

```bash
python3 build.py
```

Deployed via GitHub Pages.
